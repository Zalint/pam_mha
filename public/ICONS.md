# Icônes PWA

Pour que la PWA fonctionne correctement, vous devez ajouter deux fichiers d'icônes dans le dossier `public/` :

1. **icon-192.png** - Icône 192x192 pixels
2. **icon-512.png** - Icône 512x512 pixels

## Création des icônes

### Option 1 : Utiliser un générateur en ligne

1. Aller sur https://www.favicon-generator.org/ ou https://realfavicongenerator.net/
2. Télécharger un logo ou une image (de préférence avec fond vert institutionnel #2d8b6d)
3. Générer les icônes PWA
4. Télécharger les fichiers 192x192 et 512x512
5. Les renommer en `icon-192.png` et `icon-512.png`
6. Les placer dans le dossier `public/`

### Option 2 : Créer avec un logiciel de graphisme

- Utiliser Photoshop, GIMP, Canva, etc.
- Créer deux images carrées : 192x192 et 512x512
- Fond : vert institutionnel (#2d8b6d)
- Motif : Logo MHA ou symbole eau/hydraulique
- Exporter en PNG avec transparence si possible

### Option 3 : Icône simple avec texte

Créer une image simple avec :
- Fond vert (#2d8b6d)
- Texte blanc "MHA" ou "PAM" au centre
- Police sans-serif, gras

## Temporaire : Icône de remplacement

En attendant, vous pouvez utiliser une icône monochrome simple ou laisser le navigateur utiliser l'icône par défaut.

L'application fonctionnera sans icônes, mais l'expérience PWA sera améliorée avec des icônes appropriées.
