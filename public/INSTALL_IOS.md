# 📱 Installation sur iPhone/iPad (iOS)

## 🎯 Étapes d'Installation

### 1️⃣ Ouvrir Safari
⚠️ **Important** : Vous DEVEZ utiliser **Safari** (pas Chrome, Firefox ou autre)

### 2️⃣ Aller sur l'application
Tapez l'adresse : `https://votre-domaine.com`

### 3️⃣ Appuyer sur le bouton Partager
Cherchez l'icône **📤 Partager** en bas de l'écran Safari

### 4️⃣ Sélectionner "Sur l'écran d'accueil"
- Faites défiler le menu vers le bas
- Appuyez sur **"Sur l'écran d'accueil"** (ou "Add to Home Screen" en anglais)

### 5️⃣ Personnaliser (optionnel)
- Le nom par défaut est "Suivi PAM MHA"
- Vous pouvez le modifier si vous le souhaitez

### 6️⃣ Appuyer sur "Ajouter"
L'icône apparaîtra sur votre écran d'accueil ! 🎉

---

## ✅ Après Installation

L'application se comportera comme une vraie application native :
- ✅ Icône sur l'écran d'accueil
- ✅ Ouverture en plein écran (sans barre Safari)
- ✅ Fonctionne hors ligne (après la première visite)
- ✅ Notifications (si activées)

---

## ❓ Problèmes Courants

### Je ne vois pas "Sur l'écran d'accueil"
- Vérifiez que vous utilisez **Safari** (pas un autre navigateur)
- Mettez à jour iOS à la dernière version

### L'icône ne s'affiche pas
- Rafraîchissez la page et réessayez
- Vérifiez votre connexion internet

### L'application ne s'ouvre pas en plein écran
- Supprimez l'icône de l'écran d'accueil
- Réinstallez-la en suivant les étapes ci-dessus
